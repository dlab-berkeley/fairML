Chapter 1 Fairness and Bias in Machine Learning Workshop

This workshops provides a gentle introduction to the fairness and Bias in machine Learning applications with a focus on the ProPublica’s Analysis of the COMPAS algorithm. We revised the ProPublica’s original R and Python code to increase its code readability, blend it with other references, and published and deployed the revised notebook using bookdown and GitHub page.
